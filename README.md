## 🔍 How rag-doctor Works

rag-doctor checks your AI projects for common errors. It focuses on these areas:

- **Missing retrieval:** Finds if your AI system skips a retrieval step needed to fetch data.

- **Chunking problems:** Looks for wrong ways data is divided, which can reduce quality.

- **Embedding mismatches:** Checks if the embeddings (data representations) match across components.

- **Vector database use:** Detects bad setups in the vector database used for searching.

These checks help catch architecture flaws before they cause bigger problems.
